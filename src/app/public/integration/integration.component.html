Integration

Intégration </JS>

------------

L'intégration de PayPik sur votre site web est incroyablement simple, nécessitant seulement deux lignes de code, tout aussi facile que l'implémentation de Google Analytics. Cette simplicité d'intégration vous permet d'ajouter rapidement et efficacement les fonctionnalités de PayPik à votre site, sans avoir besoin de compétences techniques étendues.

Code JS Exemple: 

    <script id="paypik" 
        src="./widget1.js"
        data-marchand_id="5210" 
        data-access_key="eadbfc7e23ec00" 
        data-order_id="111" 
        data-amount="1000.0" 
        data-currency="MAD" 
        data-hmac=""
        data-redirect_url="">
    </script>
